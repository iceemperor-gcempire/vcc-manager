# VCC Manager 기능 인벤토리 — 새 디자인(v2) 브리프 초안

- **작성**: 2026-06-12, 전면 검토·UI 감사에서 파악한 실제 코드 기준
- **목적**: v1 디자인안의 실패 원인("앱 기능과 다른 디자인")을 반복하지 않기 위해, **실제 존재하는 기능·데이터·플로우만**을 디자인 입력으로 제공
- **원칙**: 이 문서에 없는 기능은 디자인하지 않는다. 추가하고 싶은 기능은 별도 "신규 기능 후보" 섹션에서만.

---

## 0. 시스템 골격 (디자인이 바꿀 수 없는 사실)

- 인증: 이메일/비밀번호 + Google OAuth. **가입 승인제** (pending/approved/rejected — 승인 대기 화면 필요)
- 권한 2단: 일반 사용자 / 관리자(isAdmin). 작업판은 **그룹 단위 접근 제어**
- 생성 흐름: 작업판 선택 → 입력 폼(작업판이 정의한 customField 동적 렌더) → **백그라운드 큐 처리** → 작업 히스토리에서 결과 확인. 생성 완료를 페이지에서 기다리는 흐름 아님 (폴링 기반, WebSocket 없음)
- provider: ComfyUI(이미지/영상, LoRA·시드·워크플로우) / OpenAI·호환(이미지·텍스트) / Gemini(이미지·텍스트)
- 텍스트 생성은 별도 화면(싱글턴/멀티턴 대화형) — 이미지 폼과 흐름이 다름
- 모드: 라이트(밝은 셸) / 다크(네이비 셸), 디바이스별 localStorage

## 1. 사용자 화면별 기능 인벤토리

### /dashboard 대시보드
- 인사말 + 날짜 히어로, 빠른 액션(새 프로젝트/작업판 실행/이미지 업로드)
- 위젯: 실행 중 파이프라인(폴링), 주간 생성 추이(7일 sparkline + 오늘/평균/피크), 서버 상태 목록(헬스 dot), 최근 작업 이미지 그리드, 자주 쓰는 작업판, 업데이트 로그 다이얼로그
- 데이터: dashboard API 3종(stats/recent/usage) + queue stats

### /workboards 작업판 카탈로그
- 검색(이름·설명) + 2축 필터(출력 image/video/text × 서버 ComfyUI/OpenAI/Gemini) + 카운트
- 카드: 아이콘, 이름, 타입 mono 메타(서버종류·버전), 설명, 서버 dot + 사용횟수, 갱신일, 상세정보 다이얼로그
- 프로젝트 컨텍스트 모드(?projectId= 진입 시 칩 표시)
- 클릭 → 이미지·영상 작업판은 /generate/:id, 텍스트는 /prompt-generate/:id

### /generate/:id 작업판 실행 (이미지·영상) — 사용 빈도 최상위
- 프롬프트(필수)·부정 프롬프트, 프롬프트 불러오기(저장된 프롬프트 데이터), AI 프롬프트 생성(LLM 다이얼로그)
- LoRA: ComfyUI 만 — 모달에서 선택 → 프롬프트에 `<lora:이름:가중치>` 삽입, 트리거 워드 삽입, (R2-1) 칩 표시·제거
- 시드: 무작위 토글 + 고정값 + 재생성
- 고급 설정 = 작업판이 정의한 customField 동적 렌더: select / number / text / image(참조 이미지 — 갤러리 선택+업로드, 최대 N장) / baseModel·lora(서버 모델 목록 Autocomplete + 메타데이터 picker)
- 우측: 작업판 정보(서버/버전/사용횟수/대기 큐), 생성 시작 → **즉시 /jobs 이동**
- "계속하기" 진입: 이전 작업 입력값 복원(+사용자 설정에 따라 시드 무작위화)

### /prompt-generate/:workboardId 텍스트 생성 (대화형)
- 싱글턴/멀티턴 대화(ConversationJob), 시스템 프롬프트(프롬프트 데이터 연동), 이미지 첨부(비전 LLM, #517), 스트리밍 응답(SSE), 대화 이어가기, 토큰 사용량 표시
- 세계관/시스템 프롬프트 태그 연동

### /jobs 작업 히스토리 (통합 피드)
- 세그먼트: 전체/파이프라인/이미지/영상/텍스트 + 카운트, 검색, 폴링 갱신
- 행: 타입·상태 칩(대기/진행+%/완료/실패/취소), 썸네일, 메타(모델·크기·N장), 상대시각
- 액션: 계속하기(같은 작업판 재실행), **다른 작업판으로 이어가기**(필드 매칭), 상세 다이얼로그(입력값/결과/비용·토큰), 프롬프트 저장, 재시도, 취소, 삭제(콘텐츠 동반 삭제 옵션), 텍스트는 전문 보기/이어가기
- 파이프라인 행: 단계 진행 dot + 인라인 확장

### /content 내 컨텐츠 (자산 라이브러리)
- 탭 5: 생성된 이미지/업로드된 이미지/생성된 동영상/직접 작성 텍스트/생성된 텍스트 (+카운트)
- 좌측 필터 레일: 프로젝트별, 태그별 (데스크탑)
- 그리드 카드: 썸네일, 파일명, 크기·용량·날짜, 다운로드, 뷰어(이미지/비디오 시네마 다이얼로그)
- 선택 모드: 다중 선택 → 일괄 삭제(활성 작업 참조 보호)/태그 부여/프로젝트 이동, 업로드(참조용)
- 항목 편집: 태그, 프로젝트 연결, 즐겨찾기 없음(미구현 — v1 안의 즐겨찾기는 가짜였음)

### /projects 프로젝트
- 목록: 그리드/리스트 토글, 검색, 카드(그라데이션/커버, 태그 칩, 통계, 상대시각), 생성/편집 다이얼로그
- /projects/:id 상세: 커버 헤더, 탭(개요/이미지/동영상/텍스트/작업판/파이프라인), 프로젝트 단위 자산 모음, **파이프라인 빌더/실행**(단계=작업판 체인, 텍스트→이미지 파이프, 실행 히스토리, 단계 진행)

### /prompt-data 프롬프트 데이터
- 저장된 프롬프트 CRU D, 카드(이름/프롬프트 미리보기/대표 이미지), 검색, 사용 횟수, 작업판 실행에서 불러오기와 연동

### /tags 태그 관리
- 태그 CRUD(이름/색), 사용 횟수, 빌트인 태그(세계관/시스템 프롬프트 — 삭제 불가), 태그별 콘텐츠 탐색

### /profile, /settings
- 프로필: 닉네임/비밀번호 변경, **API Key 발급·관리(최대 10개, MCP 연동용)**, 사용 통계
- 설정: 다크모드(시스템/라이트/다크), 히스토리↔콘텐츠 연동 삭제 2종, 계속하기 시 무작위 시드, NSFW LoRA/이미지 필터 2종, 작업판 출력형식 초기화 2종

### 전역
- ⌘K 명령 팔레트(페이지 이동), 알림 popover, 검색 아이콘, 사이드바(메뉴+카운트 없음/관리자 섹션), 업데이트 로그

## 2. 관리자 화면

- **/admin/dashboard**: 통계 카드(사용자/작업판/서버/작업 수), 큐 상태, 최근 가입(승인 대기 강조 필요 — 현재 약함)
- **/admin/users**: 사용자 목록 테이블, 승인(pending→approved/rejected), 관리자 토글, 그룹 배정
- **/admin/workboards**: 카탈로그(공유 컴포넌트) + 상태 필터(게시/보관), 가져오기/내보내기(JSON), 복제, 활성 토글
- **/admin/workboards/:id/edit** 작업판 편집기: 기본정보 / **입력 양식 빌더**(customField 정의: 타입·라벨·옵션·기본값·역할(role) 매핑·드래그 정렬) / ComfyUI 워크플로우 JSON + `{{##변수##}}` 매핑 / 접근 그룹 — **#440: 화면이 빽빽해 개편 대기 중이었음. 새 디자인의 최대 난제 화면**
- **/admin/servers**: 서버 카드(타입/URL/상태/응답시간), 등록·수정(API 키), 헬스체크(개별/전체), 모델 동기화
- **/admin/models**: 모델/LoRA 메타데이터 관리(Civitai 연동, 썸네일·트리거워드·NSFW 플래그)
- **/admin/groups**: 그룹 CRUD + 작업판 접근 매핑
- **/admin/stats**: 시스템 통계(사용자별/작업판별 사용량)
- **/admin/backup**: 백업 생성(시간당 1회, 진행 중 쓰기 잠금)/목록/다운로드(signed URL)/삭제, 복원(ZIP 업로드→검증→실행)

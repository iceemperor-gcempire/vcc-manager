# 작업 일지 (Work Logs)

이 폴더는 VCC Manager 프로젝트의 작업 일지를 기록합니다.

## 목적

- 사용자 요청 사항 및 구현 내용 기록
- 발생한 이슈와 해결 방법 문서화
- 변경 사항 추적 및 히스토리 관리

## 파일 명명 규칙

```
YYYY-MM-DD.md
```

예: `2026-02-04.md`

## 일지 구조

각 일지는 다음 섹션을 포함합니다:

1. **요청 사항** - 사용자가 요청한 작업 내용
2. **완료된 작업** - 구현된 기능 및 변경 사항
3. **발생한 이슈 및 해결** - 문제 상황과 해결 방법
4. **관련 커밋** - 작업과 관련된 Git 커밋
5. **다음 작업** - 예정된 후속 작업 (선택적)

# v2 업데이트 내역

## v2.0.3

### 수정
- fix: ComfyUI workflow injection 의 partial-match 경로 double-escape 회귀 (#299)
  - \`replaceInObject\` 의 부분 일치 분기가 \`escapeForJsonString\` 으로 backslash 를 \`\\\\\` 로 만든 후 JS object 에 저장. 이후 axios 가 JSON.stringify 로 한 번 더 escape — ComfyUI 가 double-escape 된 경로로 수신해 Windows 모델 경로 손상
  - 수정: partial-match 도 full-match 와 동일하게 raw value 사용. JSON.stringify 의 자동 escape 만 적용
  - fallback string-replacement (workflowString 직접 조작) 는 JSON 텍스트 컨텍스트라 escape 유지

  - referenceImageMethod → reference_image_method

- chore: \`WELL_KNOWN_FIELD_NAME_TO_ROLE\` 에 snake_case alias 추가 — 사용자 정의 신규 작업판이 snake_case 만으로도 동작

# v3 업데이트 내역

## v3.2.1

- 작업 히스토리에서 종류(전체 / 파이프라인 / 이미지 / 영상 / 텍스트)를 고르거나 검색한 뒤, 계속하기·다른 작업으로 이동했다가 **뒤로가기로 돌아오면 선택이 초기화되던 문제를 고쳤습니다** (#458). 이제 골라둔 종류와 검색어가 그대로 유지됩니다.

## v3.2.0

대시보드 첫 화면을 새로 단장하고, 작업 히스토리를 한 곳에서 보는 통합 피드로 바꿨습니다.

### 대시보드 새단장
- 로그인 직후 첫 화면이 새로워졌습니다. 인사말과 빠른 액션(새 프로젝트 / 작업판 실행 / 파이프라인 만들기 / 이미지 업로드) 아래로 한눈에 들어오는 위젯들이 배치됩니다.
- **실행 중 파이프라인**(진행률), **최근 프로젝트**, **최근 생성 이미지**, **주간 생성 추이**, **서버 상태**, **자주 쓰는 작업판**을 카드로 모아 보여줍니다.

### 작업 히스토리 — 한 곳에서 모아 보기
- 이미지 · 영상 · 텍스트 · 파이프라인 생성 기록을 **하나의 피드**에서 최신순으로 봅니다. 더 이상 탭을 오가지 않아도 됩니다.
- 상단에서 **종류별(전체 / 파이프라인 / 이미지 / 영상 / 텍스트)로 걸러보고**, 검색으로 빠르게 찾을 수 있습니다.
- 각 기록은 종류에 맞게 보입니다 — 이미지·영상은 미리보기 썸네일과 **계속하기 / 다른 작업**, 텍스트는 본문 미리보기, 파이프라인은 단계 진행 상태.

## v3.1.0

파이프라인·작업판 사용성을 다듬고, 다크모드에서 색이 튀던 곳을 정리했습니다. AI 에이전트가 내 파이프라인을 직접 실행할 수 있는 연동 기능도 새로 들어왔습니다.

### 파이프라인 단계에서 저장한 프롬프트 바로 불러오기
- 파이프라인 단계를 편집할 때, 미리 저장해 둔 프롬프트 데이터(프롬프트 / 네거티브 / 시드)를 버튼 한 번으로 불러와 채울 수 있습니다. 자주 쓰는 프롬프트를 매번 다시 입력하지 않아도 됩니다.
- 해당 작업판에 프롬프트·네거티브·시드 입력칸이 없으면 버튼이 비활성화되고 이유를 안내합니다.

### 작업판 편집기가 전체 페이지로 (관리자)
- 작업판을 새로 만들고 편집하는 화면이 팝업창에서 전체 페이지로 바뀌었습니다. 화면을 더 넓게 쓸 수 있고, 주소(URL)로 편집 화면에 바로 접근할 수 있습니다.
- 저장하지 않은 변경이 있는 채로 화면을 벗어나려 하면 경고로 알려줍니다.

### 다크모드 다듬기
- 다크모드에서 일부 영역 배경이 하얗게 떠 보이거나, 로그인·회원가입 안내 문구와 작업판 편집기의 표 등에서 색이 튀던 부분을 정리했습니다. 어두운 화면에서도 한결 일관되게 보입니다.

### AI 에이전트용 파이프라인 도구 (MCP)
- 연결된 AI 에이전트가 내 프로젝트의 파이프라인을 직접 조회하고 실행할 수 있는 도구가 추가됐습니다. 프로젝트·파이프라인 목록 조회부터 파이프라인 실행, 실행 상태·결과 확인까지 한 흐름으로 이어집니다.

- **디바이스별 영속** — 백엔드 user preferences 동기화 안 함. 기기마다 모드 선호 다를 수 있다는 사용자 정책 반영.

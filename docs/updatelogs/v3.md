### 디자인 시스템 v1 (Phase 0~4) — 토큰 / 폰트 / 사이드바 / size prop 정리

- **Phase 1 vccTheme**: `frontend/src/theme.js` 단일 진입점. Iris primary (#5B5BD6), warm-neutral 배경 (#F7F7F4), navbar palette 신설, radius 6/8, 4단계 shadow, defaultProps 일괄 size:small, 한국어 letter-spacing −0.005em
- **Phase 2 사이드바/헤더**: hardcoded hex 6종 (#2c3e50/#34495e/#ecf0f1/#bdc3c7) → `palette.navbar.*` 토큰화
- **Phase 3 size 정리**: 코드모드로 Button/Chip/TextField 의 명시 `size="small"` 229곳 제거 (theme defaultProps 대체). IconButton/FormControl/Switch/ToggleButtonGroup 등 defaultProps 없는 컴포넌트는 의도적으로 유지
- **Phase 4 한국어 가독성**: `index.css` Pretendard fontFamily + ss06/ss07 font-feature-settings + `.MuiTypography-body1/body2` text-wrap pretty

### 글로벌 UX (Phase 6)
- **⌘K 명령 팔레트** (`CommandPalette.js`): 글로벌 ⌘K/Ctrl+K + 헤더 🔍 버튼. 그룹 — 프로젝트 / 작업판 / 명령 (대시보드·내 컨텐츠·프로필·작업 히스토리·관리자 패널). 키보드 nav (↑/↓/Enter/Esc + scrollIntoView). useQuery staleTime 60s.
- **알림 popover** (`NotificationsPopover.js`): 헤더 🔔 + MUI Popover. 영속 저장 없음 — react-query 캐시의 `pipelineRuns` 키 source. 진행 중 / 이전 그룹화, unread bg 강조. 클릭 시 해당 프로젝트로 이동.

### 다크모드 (Phase 7)
- `buildVccTheme(mode)` 함수 — light/dark palette 양쪽. 디자이너 핸드오프 토큰 (dark: 배경 #0E1015, Paper #161A22, primary #7676E0, text #E8E9EE 등) 적용. hardcoded hex (navbar/divider/outlined notchedOutline) → theme.palette 참조 함수형 styleOverrides 전환.
- `ColorSchemeContext` — mode (`light`/`dark`/`system`), matchMedia(prefers-color-scheme) 구독, localStorage 영속, html `data-theme` attribute 동기화.
- **디바이스별 영속** — 백엔드 user preferences 동기화 안 함. 기기마다 모드 선호 다를 수 있다는 사용자 정책 반영.
- Header 사용자 메뉴 3-row 토글 (시스템/라이트/다크).

사전 준비 · 폰트 + 의존성

파괴적 변경 없음 예상 30분 PR #1
Pretendard와 JetBrains Mono를 index.html에 추가합니다. 기존 Roboto는 fallback으로 유지하므로 미적용 화면도 깨지지 않습니다.
+ Add public/index.html <head>
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/
       orioncactus/pretendard@v1.3.9/
       dist/web/variable/
       pretendardvariable.min.css"/>
<link rel="stylesheet"
  href="https://fonts.googleapis.com/
       css2?family=JetBrains+Mono:
       wght@400;500&display=swap"/>
+ Add CSS reset (선택)
/* 이전 코드와 충돌하지 않습니다 */
:root {
  --font-sans: "Pretendard",
    "Pretendard Variable",
    "Noto Sans KR", Roboto,
    sans-serif;
  --font-mono: "JetBrains Mono",
    "SF Mono", Menlo, monospace;
}
체크리스트
  • Pretendard CDN 로딩 확인 (Network 탭)
  • JetBrains Mono 로딩 확인 — 코드/시드/ID 영역에 자동 적용
  • QA 영역: 한국어 본문에 시각적 차이 없음 (MUI typography가 아직 Roboto 우선)

토큰 교체 · vccTheme 적용

핵심 예상 1일 PR #2 시각적 변화 큼
theme.ts를 새 vccTheme ThemeOptions로 교체합니다. 모든 색상·간격·radius·shadow가 한 번에 새 토큰으로 매핑되며, 컴포넌트 코드는 손대지 않습니다.
- Before App.js
const theme = createTheme({
  palette: {
    primary:   { main: '#1976d2' },
    secondary: { main: '#dc004e' },
  },
});
+ After App.js
import { vccTheme } from './theme';
const theme = createTheme(vccTheme);
파급 효과
  • Primary 컬러가 #1976d2 → #5B5BD6 (Iris) — system-prompt 태그와 시각적 구별됨
  • Background가 #FFFFFF → #F7F7F4 — warm-neutral 토널
  • radius가 4 → 6 기본, 카드는 8
  • Button defaultProps size: 'small' 적용 — 312곳의 명시 props 삭제 후속 가능 (다음 PR)
  • QA 영역: AppBar / Sidebar는 아직 hex hardcode 상태 — Phase 2에서 정리

사이드바 / 헤더 통합

핵심 예상 4시간 PR #3
흩어진 #2c3e50 / #34495e / #ecf0f1 / #bdc3c7 hex 6종을 모두 theme.palette.navbar 토큰 패밀리로 교체합니다. Sidebar.js / Header.js 2 파일에 집중.
- Before Sidebar.js (8 occurrences)
sx={{ backgroundColor: '#2c3e50' }}
sx={{ '&.active': { bg: '#34495e' }}}
sx={{ color: '#ecf0f1' }}
sx={{ color: '#bdc3c7' }}
+ After
sx={{ bgcolor: 'navbar.main' }}
sx={{ '&.active': { bgcolor: 'navbar.light' }}}
sx={{ color: 'navbar.contrastText' }}
sx={{ color: 'grey.500' }}
체크리스트
  • grep -rn '#2c3e50\|#34495e\|#ecf0f1\|#bdc3c7' src/ 결과 0줄
  • 사이드바 활성 항목 좌측에 2px Iris 액센트 바 추가 (CSS-in-JS)
  • 관리자 메뉴 좌측 빨강 액센트는 error.main 토큰으로 유지
  • QA: 다크모드 미사용 상태에서도 사이드바 색이 자연스럽게 더 정돈된 슬레이트로 변경됨

컴포넌트 defaultProps 정리

예상 3시간 PR #4 코드 축소
Button size="small" 312곳, Chip size="small" variant="outlined" 다수, TextField size="small"가 모두 defaultProps로 흡수됩니다. 매번 명시할 필요 없음.
- Before 312 곳
<Button size="small" variant="contained">저장</Button>
<Chip size="small" variant="outlined" />
<TextField size="small" variant="outlined" />
+ After 312 → 0 props 명시
<Button variant="contained">저장</Button>
<Chip />
<TextField />
자동화 가능
  • codemod: jscodeshift로 size="small" prop 제거 자동화
  • PR 크기는 크지만 의미 변경 없음 — 리뷰는 빠르게
  • QA: medium size 의도적으로 사용한 11곳은 유지
4

한국어 가독성 + 폰트

예상 2시간 PR #5
typography.fontFamily를 Pretendard 우선으로 명시. 한국어 letter-spacing(-0.005em), text-wrap: pretty를 본문에 적용해 줄바꿈 자연스럽게.
App.css / 글로벌
body {
  font-feature-settings: "ss06" on, "ss07" on;
}
.MuiTypography-body1,
.MuiTypography-body2 {
  letter-spacing: -0.005em;
  text-wrap: pretty;
}
체크리스트
  • Pretendard 9 가중치 모두 로딩 확인 (Variable 폰트)
  • 긴 한국어 라벨 (e.g. "현재 단계만 다시 실행") — wrap/ellipsis 정책 확인
  • mono 폰트가 ID/시드/타임스탬프 영역에 적용됨

핵심 페이지 — 점진 적용

예상 1주 PR #6 ~ #N
파이프라인 빌더 (novel lane variant), 파이프라인 실행 결과, 컨텐츠 라이브러리 — 화면별로 별도 PR. 각 화면은 mockup과 1:1 매칭해 시각 회귀 줄임.
권장 머지 순서
  • 5a · 프로젝트 상세 페이지 (탭 구조 정리)
  • 5b · 파이프라인 실행 + 히스토리 (Stepper · 결과 카드)
  • 5c · 컨텐츠 라이브러리 (필터 레일 + 그리드)
  • 5d · 파이프라인 빌더 (lane variant — novel UI)
  • 5e · 작업판 빌더 (admin 3-pane)

글로벌 UX — ⌘K + 알림

예상 3일 PR #N+1 신규 기능
⌘K 명령 팔레트, 알림 popover, 인증 페이지 리프레시. 백엔드 변경 없이 추가 가능. 검색은 React Query useQueries로 프로젝트/작업판/파이프라인 동시 조회.

다크 모드

예상 2일 PR #N+2 옵트인
MUI colorSchemes API로 토큰만 추가. CSS-in-JS 코드 변경 거의 없음. 사용자 설정에서 토글 가능.

      background: {
        default: '#0E1015',
        paper:   '#161A22',
      },
      // ... dark.* tokens from tokens.css
    },
  },
}

롤백 · 리스크

계획

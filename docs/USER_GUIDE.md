# VCC Manager 사용자 가이드

VCC Manager를 사용하여 AI 이미지를 생성하고 관리하는 방법을 안내합니다.

## 1. 시작하기

### 계정 생성 및 승인
1. **회원가입**: 이메일과 비밀번호로 계정을 생성하거나 Google 계정으로 가입합니다.
2. **승인 대기**: 새 계정은 관리자 승인이 필요합니다.
3. **승인 완료**: 관리자가 승인하면 모든 기능을 사용할 수 있습니다.

### 로그인
서비스에 접속하면 로그인 화면이 나타납니다.
- **이메일 로그인**: 가입한 이메일과 비밀번호로 로그인합니다.
- **구글 로그인**: Google 계정을 사용하여 간편하게 로그인합니다.

**로그인 상태별 안내:**
- **승인 대기 중**: "계정 승인 대기 중입니다" 메시지가 표시됩니다.
- **승인 거절**: "계정 승인이 거절되었습니다" 메시지가 표시됩니다.
- **승인 완료**: 정상적으로 대시보드에 접근할 수 있습니다.

### 대시보드 (개선됨)
로그인 후 대시보드에서 핵심 정보를 간결하게 확인할 수 있습니다:
- **서버 전체 작업 상태**: 현재 대기 중/처리 중인 작업 수
- **내 작업 상태**: 개인의 대기 중/처리 중인 작업 수  
- **빠른 액션**: 이미지 생성, 이미지 업로드, 작업 히스토리 보기 버튼

### 모바일 사용법 (신규)
모바일 기기에서 사용할 때:
- **햄버거 메뉴**: 좌상단의 ≡ 버튼을 눌러 메뉴에 접근
- **간소화된 제목**: 모바일에서는 "VCCM"로 표시
- **터치 친화적**: 메뉴 항목과 버튼이 터치하기 쉽게 최적화됨

## 2. 이미지 생성하기 (Workboard)

이미지를 생성하려면 **작업판(Workboard)** 메뉴로 이동하세요.

1. **작업판 선택**: 원하는 스타일이나 모델이 설정된 작업판을 선택합니다.
2. **프롬프트 입력**:
   - **Prompt**: 생성하고 싶은 이미지에 대한 설명을 영어로 입력합니다.
   - **Negative Prompt**: 이미지에 포함되지 않았으면 하는 요소를 입력합니다.
3. **설정 조절** (선택사항):
   - 이미지 크기, 스텝 수, CFG 스케일 등을 조절할 수 있습니다.
4. **생성 시작**: '생성하기' 버튼을 클릭합니다.
   - 작업은 백그라운드에서 처리되며, 다른 페이지로 이동해도 계속 진행됩니다.

## 3. 작업 히스토리 (개선됨)

**작업 히스토리** 메뉴에서 이미지 생성 작업들을 관리할 수 있습니다.

### 기본 기능
- 생성 작업들의 상태와 진행도를 실시간으로 확인
- 프롬프트 검색 기능으로 원하는 작업 빠르게 찾기
- 페이지당 정확히 10개 항목 표시 (개선됨)

### 모바일 최적화 (신규)
- 스마트 페이지네이션: 최대 3개 페이지 번호만 표시
- 반응형 그리드 레이아웃으로 메타데이터 깔끔하게 정렬
- 긴 프롬프트 자동 줄임 처리
- 터치하기 쉬운 버튼 크기와 간격

### 향상된 페이지네이션 (신규)
작업 히스토리에서 새로운 페이지네이션 기능을 사용할 수 있습니다:
- **직접 페이지 이동**: 📝 아이콘을 클릭하여 원하는 페이지 번호를 직접 입력
- **빠른 탐색**: 처음/마지막 페이지로 한 번에 이동 가능
- **스마트 표시**: 페이지가 많아도 화면 공간을 효율적으로 사용

### 작업 관리
- **작업 삭제 (개선됨)**: 작업 삭제 시 연관된 생성 이미지도 자동 삭제
- **작업 재시도**: 실패한 작업을 다시 실행
- **작업 취소**: 대기 중/처리 중인 작업 취소
- **이미지 보기**: 생성된 이미지들을 크게 보기

## 4. 내 이미지 (개선됨)

**내 이미지** 메뉴에서 생성된 이미지와 업로드한 이미지를 관리할 수 있습니다.

### 생성된 이미지 탭
- 생성된 모든 이미지를 모아볼 수 있습니다
- 이미지 상세보기에서 다운로드 버튼 제공 (신규)
- 생성에 사용된 프롬프트와 설정을 확인할 수 있습니다

### 업로드된 이미지 탭
- Image-to-Image 등의 기능을 위해 업로드한 이미지들
- 업로드한 이미지는 작업판에서 참조 이미지로 사용 가능
- 드래그 앤 드롭으로 간편한 이미지 업로드

### 모바일에서 이미지 다운로드 (개선됨)
iPhone Safari에서 다운로드 문제가 해결되었습니다:
- **일반 브라우저**: 자동 다운로드
- **iPhone Safari**: 새 탭에서 이미지 열림 → 이미지를 길게 눌러서 저장
- 다운로드 실패 시 친화적인 오류 메시지 표시

### 향상된 탐색 기능 (신규)
내 이미지 페이지에서도 개선된 페이지네이션을 사용할 수 있습니다:
- **직접 페이지 이동**: 많은 이미지가 있을 때 원하는 페이지로 바로 이동
- **효율적인 탐색**: 처음/이전/다음/마지막 페이지 버튼으로 빠른 이동
- **모바일 최적화**: 화면 크기에 관계없이 깔끔한 페이지네이션 표시

## 5. 관리자 기능 (확장됨)

관리자 계정으로 로그인 시 **관리자 패널**에 접근할 수 있습니다.

### 사용자 관리 (신규 확장)
- **가입 승인 관리**: 새로 가입한 사용자들의 승인/거절 처리
- **사용자 목록**: 모든 사용자 조회 및 상태 관리
- **승인 상태별 필터링**: 대기 중/승인됨/거절됨 상태별 조회
- **사용자 검색**: 이메일이나 닉네임으로 사용자 검색
- **계정 삭제**: 사용자 계정 및 관련 데이터 완전 삭제

### 작업 관리
- **전체 작업 모니터링**: 모든 사용자의 작업 상태 조회
- **작업 통계**: 상태별 작업 현황 및 통계 정보
- **작업 필터링**: 사용자별, 상태별 작업 조회

### 시스템 관리
- **작업판 관리**: 새로운 워크플로우 템플릿을 생성하거나 수정
- **시스템 통계**: 전체 시스템 사용량 및 성능 지표
- **큐 상태 모니터링**: 실시간 작업 큐 상태 확인

## 6. 배포 및 운영 (신규)

### 프로덕션 서버 관리
운영진을 위한 배포 및 서버 관리 가이드:

#### 서버 시작
```bash
./deploy-prod.sh
```
- 자동 데이터베이스 백업 안내
- 안전한 컨테이너 재시작
- 헬스체크 및 상태 확인

#### 서버 중지 (개선됨)
```bash
./scripts/stop-production.sh
```
- Docker Compose를 사용한 안전한 종료
- 데이터베이스 볼륨 보존
- 상태 확인 및 안내 메시지

## 7. 문제 해결 (업데이트됨)

### 일반적인 문제들

#### 모바일에서 메뉴 접근 안 됨
**해결**: 좌상단의 햄버거 메뉴(≡) 버튼을 사용하세요.

#### iPhone Safari에서 이미지 다운로드 안 됨
**해결**: 이미지가 새 탭에서 열리면 이미지를 길게 눌러서 "이미지 저장"을 선택하세요.

#### 대시보드에 정보가 너무 많음
**해결**: v1.1.0부터 일반 사용자용 대시보드가 간소화되었습니다. 핵심 정보만 표시됩니다.

#### 작업 히스토리에서 페이지네이션이 화면을 벗어남
**해결**: v1.1.0부터 스마트 페이지네이션이 적용되어 모바일에서도 적절히 표시됩니다.

#### 원하는 페이지로 빠르게 이동하고 싶음
**해결**: 페이지네이션에서 📝 아이콘을 클릭하면 직접 페이지 번호를 입력하여 이동할 수 있습니다.

#### 내 이미지 페이지에서도 페이지가 많을 때 탐색이 어려움
**해결**: 작업 히스토리와 동일한 향상된 페이지네이션이 적용되어 효율적인 탐색이 가능합니다.

### 계정 관련 문제

#### 가입 후 로그인이 안 됨
**원인**: 계정 승인 대기 상태  
**해결**: 관리자 승인을 기다리거나 관리자에게 문의하세요.

#### 승인이 거절됨
**해결**: 관리자에게 재승인을 요청하거나 거절 사유를 문의하세요.

# LoRA  목록조회 기능 v1

comfyUI 서버를 통하여 lora 목록을 받아올 수 있는 화면을 구성했으면 합니다.

1. comfyUI 서버로부터 모든 lora 목록을 호출
	1. 처음에 lora 목록 캐시 데이터가 없다면 무조건 모든 lora 목록을 comfyUI 서버로부터 호출. 캐시가 있다면 캐시를 불러옴
	2. lora 목록은 작업판 단위로 backend 에서 캐싱.
	3. 캐시 만료기간은 없으며 유저가 갱신을 호출할 수 있음.
2. 프론트에서는 lora 목록을 리스트 형태로 조회할 수 있음.
	1. 이 lora 목록은 작업판 선택 후 별도 버튼을 눌러 확인할 수 있음.
	2. 이 목록에는 "추가" 버튼이 존재하며, 이 버튼을 눌렀을 경우, 기본 설정의 "프롬프트" 영역에 lora 형식의 문자열로 추가됨. 예를 들면 <lora:확장자없는lora파일명:1> 형태로.

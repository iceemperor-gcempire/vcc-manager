# Tools (사전 설치 도구)

VCC Manager와 함께 사용하기 위해 외부 시스템에 설치해야 하는 도구들입니다.

## 포함된 도구

### comfyui-vcc-lora-hash

ComfyUI용 커스텀 노드로, LoRA 파일의 SHA256 해시를 제공합니다.

**설치 방법:**
```bash
cp -r tools/comfyui-vcc-lora-hash /path/to/ComfyUI/custom_nodes/
```

자세한 내용은 [comfyui-vcc-lora-hash/README.md](./comfyui-vcc-lora-hash/README.md)를 참고하세요.
